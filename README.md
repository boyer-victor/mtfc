# mtfc

## Info

Static website for MTFC built with [Astro](https://astro.build) and deployed to [Cloudflare Pages](https://pages.cloudflare.com).

Pages: home, about, coaches, schedule, announcements, gallery.

## Start

Install dependencies and start the dev server:

```sh
bun install
bun dev
```

| Command        | Action                                      |
| :------------- | :------------------------------------------ |
| `bun dev`      | Start local dev server at `localhost:4321`  |
| `bun build`    | Build production site to `./dist/`          |
| `bun preview`  | Preview the production build locally        |
